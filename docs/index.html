Tasks Eye

You already know how to put tasks in Obsidian. The harder part comes later: returning to a vault full of notes and knowing which one deserves your attention now.

Tasks Eye turns those notes into one daily control panel. Each row represents a note, its earliest unchecked dated task becomes the visible next action, and its folder tells you the context. When you are unsure what to do next, press Ctrl+1 and start with Open.

Your Markdown remains the source of truth. Tasks Eye reads and updates the same checkboxes and dates you already use with the Obsidian Tasks plugin; it does not create a second task database.

How to read Open

Section titled “How to read Open”
  1. Begin with Today.It contains work due today and anything overdue, so it is the shortest path to a deliberate next decision.
  2. Read one row as one note.The bold line is the earliest unchecked dated task; the note title underneath opens the surrounding context.
  3. Let sections hide the horizon.Tomorrow, This Month, Next Month, and Future keep later work visible without letting it compete with today.
  4. Use folders as contexts.The label on the right comes from the note’s folder. Use the filter at the top when you want to focus on one part of life or work.

Think “next attention date,” not only “deadline.” A task date answers “when should this note surface again?” If today is not the right moment, moving the date is a valid decision—not a failure to finish the task.

The daily loop

Section titled “The daily loop”

The goal is not to finish every possible task. It is to handle every item that asks for attention today, then keep the system trustworthy enough to use again tomorrow.

When you need directionCtrl+1 → Open → TodayChoose the first note that needs a decision.
Do itComplete the visible next task.
Move itGive it a new attention date.
Change its stateMove the note to Hold or close it.
Return to Today until every item has been handled.Later work stays in Open, paused work waits in Hold, and completed tasks can be reviewed in Done.
The trust loopCtrl+2 → Inbox → repairGive malformed or unrouted notes a next action, a date, a context, or the right status.

Act from the board—or open the note

Section titled “Act from the board—or open the note”

If the next action is already clear, complete or reschedule it without leaving Open. If you need the decision history, meeting notes, links, or supporting material, open the note and work with the task in its full context. The board is a fast point of contact, not a replacement for the note.

Inbox keeps Open believable

Section titled “Inbox keeps Open believable”

Open is useful only when every active note has a clear next action. Inbox finds the exceptions: a missing task date, an open note with nothing left to do, an unsupported status, or a note that still needs a context folder. Repair these items until Inbox is empty, and Open stays dependable.

A useful everyday rhythm

Section titled “A useful everyday rhythm”
  1. Press Ctrl+1 and handle Today toward zero: complete, reschedule, pause, or close each item.
  2. Press Ctrl+2 and repair Inbox toward zero so tomorrow’s Open view remains trustworthy.
  3. Check Hold when you are ready to reactivate paused work, and use Done when you want to look back at what was completed.
