Feature

Daily completed summary block

Daily notes can embed a completed-task summary with the ggajos-tasks-eye-daily-completed Markdown code block.

Rationale

Why it exists.

Why The Daily Summary Block Exists

The Done view is useful for browsing, but daily notes are the long-term journal of what actually happened. The block lets completed work become part of that daily record without copying task text by hand.

The aliases exist so older notes keep rendering while the project settles on the canonical block name.

Executable documentation

Acceptance criteria.

  • The canonical daily-completed code block renders completed tasks for the note date.
  • Legacy block names remain supported as migration aliases.
  • The block derives the date from source content or the daily note filename.
  • The rendered block includes previous and next daily-note navigation links when available.
