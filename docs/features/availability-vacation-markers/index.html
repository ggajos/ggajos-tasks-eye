Feature

Vacation availability markers

Open boards show configured weekends, holidays, and custom OOO dates as availability markers through the work timeline.

Rationale

Why it exists.

Why Vacation Markers Exist

Due dates are only useful if they reflect real availability. Tasks Eye shows OOO days directly in the board so scheduled work can be judged against weekends, holidays, and custom unavailable dates.

The markers are visible in Open because that is where scheduling decisions are made. The dedicated OOO filter isolates availability when the user wants a calendar-like review.

Executable documentation

Acceptance criteria.

  • Configured weekends, annual holidays, movable holidays, and custom dates are recognized as unavailable.
  • Open boards interleave vacation markers with dated work.
  • The OOO filter shows only availability markers.
  • Normal context filters suppress availability markers.
