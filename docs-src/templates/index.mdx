Tasks Eye turns Markdown notes and Tasks checkboxes into a note-centered work
view. Open shows active notes, Inbox surfaces notes that need cleanup, Hold keeps
backlog and paused work visible, and Done reviews completed tasks by date.

## Getting started

1. Install and enable the Obsidian Tasks community plugin.
2. Choose the notes folder in Tasks Eye settings; it defaults to the vault root (<code>/</code>).
3. Use <code>status: open</code>, <code>hold</code>, <code>closed</code>, or <code>archived</code> in note frontmatter.
4. Open Tasks Eye from its ribbon icon or a keyboard shortcut.

## Workflow

<ol class="workflow-list">
  <li><strong>Open</strong><span>Review active notes by their next due task.</span></li>
  <li><strong>Inbox</strong><span>Clean up notes with missing or inconsistent workflow data.</span></li>
  <li><strong>Hold</strong><span>Keep backlog and paused notes visible outside active work.</span></li>
  <li><strong>Done</strong><span>Review tasks completed on a selected date.</span></li>
</ol>
